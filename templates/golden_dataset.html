About Golden Dataset

The Golden Dataset contains verified, high-quality matches that are used to improve future AI searches. These examples help the AI:

  • Reduce hallucinations - AI learns from verified examples
  • Improve accuracy - Better pattern recognition for similar organizations
  • Fill gaps - Suggest similar organizations when primary sources miss them
  • Maintain consistency - Standardized format and quality across searches

How to use: Mark excellent results as "Golden" using the ‚≠ê button in search results. These will automatically be used to improve future searches.
